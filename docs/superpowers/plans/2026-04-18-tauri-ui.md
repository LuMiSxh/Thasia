# Tauri UI Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a Tauri 2 desktop app with SvelteKit frontend to the Thasia workspace — a 9-step modular wizard with a context-aware collapsible sidebar, a memory-efficient page editor (custom `thasia://` URI protocol, zero image bytes in JS), and full specta-generated TypeScript bindings.

**Architecture:** `src-tauri/` lives alongside the workspace (own `Cargo.lock`, separate Tauri workspace) and references the `crates/*` via path dependencies with the `tauri` feature flag on `thasia-core`. The frontend is a SvelteKit static SPA in `src/`. A custom `thasia://image?path=` URI scheme handler in Rust serves images on demand so the page editor never copies pixel data through IPC.

**Tech Stack:** Tauri 2.10, tauri-specta =2.0.0-rc.21, SvelteKit 2 + Svelte 5, Tailwind CSS v4, pnpm, TypeScript 5, `@sveltejs/adapter-static`.

**Files:**

The protocol serves any file by absolute path encoded in the query string. This means JS never holds image bytes — only `src="thasia://image?path=..."` URL strings.

- [ ] **Step 1: Write the protocol handler**

Replace `src-tauri/src/protocol.rs`:

```rust
use std::path::PathBuf;
use tauri::http::{Request, Response};

/// Build a `thasia://image?path={url_encoded_path}` URL for a given absolute path.
pub fn image_url(absolute_path: &std::path::Path) -> String {
    let encoded = urlencoding::encode(absolute_path.to_string_lossy().as_ref()).into_owned();
    format!("thasia://image?path={encoded}")
}

/// Synchronous handler registered with tauri::Builder::register_uri_scheme_protocol.
/// Reads the file at the decoded path and returns it with the appropriate MIME type.
pub fn handle(request: Request<Vec<u8>>) -> Response<Vec<u8>> {
    let uri = request.uri().to_string();

    // Extract the `path` query parameter
    let path_str = match extract_path_param(&uri) {
        Some(p) => p,
        None => return error_response(400, "missing path param"),
    };

    let path = PathBuf::from(&path_str);
    if !path.exists() {
        return error_response(404, "file not found");
    }

    let bytes = match std::fs::read(&path) {
        Ok(b) => b,
        Err(_) => return error_response(500, "read error"),
    };

    let mime = mime_for_path(&path);

    Response::builder()
        .status(200)
        .header("Content-Type", mime)
        .header("Access-Control-Allow-Origin", "*")
        .body(bytes)
        .unwrap()
}

fn extract_path_param(uri: &str) -> Option<String> {
    // URI looks like: thasia://image?path=%2Ftmp%2Ffoo.jpg
    let query = uri.split('?').nth(1)?;
    for pair in query.split('&') {
        if let Some(val) = pair.strip_prefix("path=") {
            return urlencoding::decode(val).ok().map(|s| s.into_owned());
        }
    }
    None
}

fn mime_for_path(path: &std::path::Path) -> &'static str {
    match path.extension().and_then(|e| e.to_str()).map(|e| e.to_lowercase()).as_deref() {
        Some("jpg") | Some("jpeg") => "image/jpeg",
        Some("png") => "image/png",
        Some("webp") => "image/webp",
        Some("avif") => "image/avif",
        _ => "application/octet-stream",
    }
}

fn error_response(status: u16, msg: &str) -> Response<Vec<u8>> {
    Response::builder()
        .status(status)
        .body(msg.as_bytes().to_vec())
        .unwrap()
}

    path: String,
    state: State<'_, RwLock<ConvState>>,
) -> Result<Vec<VolumeMeta>, String> {
    let source_path = PathBuf::from(&path);

    // Build the source — handles both directories and ZIP/CBZ archives
    let source = if LocalSource::is_archive(&source_path) {
        LocalSource::from_archive(source_path).await.map_err(|e| e.to_string())?
    } else {
        LocalSource::new(source_path)
    };

    let resolver = Resolver::new(RuleConfig::default());

    // Discover all images
    let mut rx = {
        use thasia_source::Source;
        source.discover().await.map_err(|e| e.to_string())?
    };

    // Parse each discovered image
    let mut parsed_images = Vec::new();
    while let Some(img) = rx.recv().await {
        match resolver.resolve(img) {
            Ok(parsed) => parsed_images.push(parsed),
            Err(e) => tracing::warn!("Skipping unresolved image: {e}"),
        }
    }

    // Group by volume number (default to volume 1 if none detected)
    let mut volume_map: BTreeMap<u32, Vec<_>> = BTreeMap::new();
    for parsed in parsed_images {
        let vol = parsed.identifier.volume.unwrap_or(1);
        volume_map.entry(vol).or_default().push(parsed);
    }

    // Sort pages within each volume by page number
    for pages in volume_map.values_mut() {
        pages.sort_by_key(|p| p.page_number);
    }

    // Build VolumeMeta for the frontend — only paths encoded as URLs, no bytes
    let result: Vec<VolumeMeta> = volume_map
        .iter()
        .map(|(vol_num, pages)| VolumeMeta {
            volume_num: *vol_num,
            pages: pages
                .iter()
                .enumerate()
                .map(|(page_index, p)| PageMeta {
                    volume_num: *vol_num,
                    page_index,
                    url: image_url(&p.source.absolute_path),
                    file_name: p.source.absolute_path
                        .file_name()
                        .and_then(|n| n.to_str())
                        .unwrap_or("")
                        .to_string(),
                })
                .collect(),
        })
        .collect();

    // Store scan result + keep source alive (holds TempDir for ZIP extractions)
    {
        let scan_data: Vec<(u32, Vec<_>)> = volume_map.into_iter().collect();
        let mut s = state.write().map_err(|e| e.to_string())?;
        s.scan_result = Some(scan_data);
        s.source = Some(source);
    }

    Ok(result)
}

use crate::state::{BundleMode, ConvState, ConvertOptions, VolumeEdit};
use std::collections::BTreeMap;
use std::path::PathBuf;
use std::sync::RwLock;
use std::time::Instant;
use tauri::{AppHandle, State};
use tauri_specta::Event;
use thasia_core::models::{ImageFormat, ParsedImage};
use thasia_processor::EncodeOptions;
use thasia_packager::{CbzGenerator, EpubGenerator, Generator, RawGenerator};
use thasia_source::{LocalSource, Source};
use std::sync::Arc;

#[tauri::command]
#[specta::specta]
pub async fn convert(
    options: ConvertOptions,
    edits: Vec<VolumeEdit>,
    state: State<'_, RwLock<ConvState>>,
    app: AppHandle,
) -> Result<(), String> {
    let start = Instant::now();

    // Pull the scan result out of state
    let scan_result = {
        let s = state.read().map_err(|e| e.to_string())?;
        s.scan_result.clone().ok_or("No scan result — run scan_source first")?
    };

    // Build a lookup: volume_num → pages
    let scan_map: BTreeMap<u32, Vec<ParsedImage>> = scan_result.into_iter().collect();

    // Apply page edits to produce final per-volume page lists
    let mut volumes: Vec<(u32, Vec<ParsedImage>, String)> = Vec::new();
    let total_volumes = edits.len() as u32;

    for edit in &edits {
        let original_pages = scan_map.get(&edit.volume_num).cloned().unwrap_or_default();
        let mut final_pages: Vec<ParsedImage> = Vec::new();

        for entry in &edit.pages {
            if entry.excluded {
                continue;
            }
            if let Some(idx) = entry.original_page_index {
                if let Some(p) = original_pages.get(idx) {
                    final_pages.push(p.clone());
                }
            } else if let Some(ref cp) = entry.custom_path {
                // Custom image: create a synthetic ParsedImage
                let path = PathBuf::from(cp);
                let rel = path.file_name().unwrap_or_default().to_string_lossy().into_owned();
                final_pages.push(thasia_core::models::ParsedImage {
                    source: thasia_core::models::DiscoveredImage {
                        absolute_path: path,
                        relative_path: rel,
                    },
                    identifier: thasia_core::models::ChapterIdentifier {
                        volume: Some(edit.volume_num),
                        chapter: None,
                    },
                    page_number: final_pages.len() as u32,
                    is_cover: final_pages.is_empty(),
                });
            }
        }

        let vol_name = if total_volumes == 1 && options.hide_single_volume {
            options.output_name.clone()
        } else {
            format!("{}{}{}", options.output_name, options.volume_separator, edit.volume_num)
        };

        volumes.push((edit.volume_num, final_pages, vol_name));
    }

    // Determine output root
    let out_root = if options.create_directory {
        PathBuf::from(&options.output_dir).join(&options.output_name)
    } else {
        PathBuf::from(&options.output_dir)
    };
    tokio::fs::create_dir_all(&out_root).await.map_err(|e| e.to_string())?;

    let encode_opts = Arc::new(EncodeOptions {
        format: options.image_format.clone(),
        max_width: options.max_width,
    });

    let mut successful = 0u32;
    let mut failed = 0u32;

    for (vol_num, pages, vol_name) in volumes {
        VolumeStartEvent {
            volume_num: vol_num,
            volume_name: vol_name.clone(),
            total_volumes,
        }
        .emit(&app)
        .ok();

        // Build a single-use LocalSource from the page list paths
        let result = convert_volume(
            pages,
            vol_name.clone(),
            &out_root,
            &options,
            encode_opts.clone(),
            vol_num,
            &app,
        )
        .await;

        match result {
            Ok(()) => {
                successful += 1;
                VolumeCompleteEvent { volume_num: vol_num, success: true, error: None }
                    .emit(&app)
                    .ok();
            }
            Err(e) => {
                failed += 1;
                tracing::error!("Volume {} failed: {e}", vol_num);
                VolumeCompleteEvent {
                    volume_num: vol_num,
                    success: false,
                    error: Some(e),
                }
                .emit(&app)
                .ok();
            }
        }
    }

    ConversionCompleteEvent {
        successful,
        failed,
        duration_secs: start.elapsed().as_secs_f64(),
    }
    .emit(&app)
    .ok();

    Ok(())
}

async fn convert_volume(
    pages: Vec<ParsedImage>,
    vol_name: String,
    out_root: &PathBuf,
    options: &ConvertOptions,
    encode_opts: Arc<EncodeOptions>,
    vol_num: u32,
    app: &AppHandle,
) -> Result<(), String> {
    let total = pages.len() as u32;

    // Build a temporary LocalSource from the page paths
    let tmp = tempfile::tempdir().map_err(|e| e.to_string())?;
    let source = Arc::new(LocalSource::new(tmp.path().to_path_buf()));
    // (We pass pages directly through the channel rather than re-scanning)

    let (tx_parsed, rx_parsed) = tokio::sync::mpsc::channel(256);
    let app_clone = app.clone();
    let vn = vol_num;

    tokio::spawn(async move {
        let mut current = 0u32;
        for parsed in pages {
            let _ = tx_parsed.send(parsed).await;
            current += 1;
            ImageProgressEvent { volume_num: vn, current, total }.emit(&app_clone).ok();
        }
    });

    let mut rx_processed =
        thasia_processor::start_pipeline(source, rx_parsed, encode_opts).await;

    let mut gen: Box<dyn Generator> = match options.output_format {
        thasia_core::models::OutputFormat::Cbz => Box::new(CbzGenerator::new()),
        thasia_core::models::OutputFormat::Epub => {
            Box::new(EpubGenerator::new().with_direction(options.direction.clone()))
        }
        thasia_core::models::OutputFormat::Raw => Box::new(RawGenerator::new()),
    };

    gen.init(out_root, &vol_name).await.map_err(|e| e.to_string())?;

    while let Some(result) = rx_processed.recv().await {
        let img = result.map_err(|e| e.to_string())?;
        gen.add_page(img).await.map_err(|e| e.to_string())?;
    }

    gen.finalize().await.map_err(|e| e.to_string())?;
    Ok(())
}

    let { onNext, onBack }: { onNext: () => void; onBack: () => void } = $props();

    let activeVolumeIndex = $state(0);

    let volumes = $derived(wizard.scanResult ?? []);
    let activeVolume = $derived(volumes[activeVolumeIndex]);
    let activeEdits = $derived(wizard.pageEdits[activeVolumeIndex]?.pages ?? []);

    // Drag state
    let dragSourceIndex = $state<number | null>(null);

    function setActiveVolume(i: number) {
        activeVolumeIndex = i;
    }

    function toggleExclude(pageIndex: number) {
        const edits = wizard.pageEdits[activeVolumeIndex];
        if (!edits) return;
        edits.pages[pageIndex].excluded = !edits.pages[pageIndex].excluded;
        // Trigger reactivity
        wizard.pageEdits = [...wizard.pageEdits];
    }

    function onDragStart(i: number) {
        dragSourceIndex = i;
    }

    function onDrop(targetIndex: number) {
        if (dragSourceIndex === null || dragSourceIndex === targetIndex) return;
        const edits = wizard.pageEdits[activeVolumeIndex];
        if (!edits) return;
        const pages = [...edits.pages];
        const [moved] = pages.splice(dragSourceIndex, 1);
        pages.splice(targetIndex, 0, moved);
        edits.pages = pages;
        wizard.pageEdits = [...wizard.pageEdits];
        dragSourceIndex = null;
    }

    async function addCustomImage() {
        const selected = await open({
            filters: [{ name: 'Image', extensions: ['jpg', 'jpeg', 'png', 'webp', 'avif'] }],
            multiple: false,
            title: 'Add custom image',
        });
        if (typeof selected !== 'string') return;

        const edits = wizard.pageEdits[activeVolumeIndex];
        if (!edits) return;

        // Insert at position 0 (front) so it becomes the cover
        edits.pages.unshift({
            originalPageIndex: null,
            customPath: selected,
            excluded: false,
        });
        wizard.pageEdits = [...wizard.pageEdits];
    }

    function imageUrl(pageIndex: number): string {
        const edit = activeEdits[pageIndex];
        if (!edit) return '';
        if (edit.customPath) {
            return `thasia://image?path=${encodeURIComponent(edit.customPath)}`;
        }
        if (edit.originalPageIndex !== null && activeVolume) {
            return activeVolume.pages[edit.originalPageIndex]?.url ?? '';
        }
        return '';

        const edit = activeEdits[pageIndex];
        if (!edit) return '';
        if (edit.customPath) return edit.customPath.split('/').at(-1) ?? 'custom';
        if (edit.originalPageIndex !== null && activeVolume) {
            return activeVolume.pages[edit.originalPageIndex]?.file_name ?? '';
        }
        return '';

<script lang="ts">
    import { onMount, onDestroy } from 'svelte';
    import { wizard } from '$lib/wizard/state.svelte';
    import { commands, events } from '$types/bindings';
    import { goto } from '$app/navigation';

    let { onNext, onBack }: { onNext: () => void; onBack: () => void } = $props();

    type VolumeProgress = {
        name: string;
        current: number;
        total: number;
        done: boolean;
        success?: boolean;
    };

    let status = $state<'idle' | 'converting' | 'done' | 'error'>('idle');
    let volumes = $state<Map<number, VolumeProgress>>(new Map());
    let errorMessage = $state('');
    let duration = $state(0);

    let unlisteners: Array<() => void> = [];

    onMount(async () => {
        // Subscribe to events
        unlisteners.push(
            await events.volumeStartEvent.listen((e) => {
                volumes = new Map(volumes).set(e.payload.volume_num, {
                    name: e.payload.volume_name,
                    current: 0,
                    total: 0,
                    done: false,
                });
            })
        );
        unlisteners.push(
            await events.imageProgressEvent.listen((e) => {
                const existing = volumes.get(e.payload.volume_num);
                if (existing) {
                    volumes = new Map(volumes).set(e.payload.volume_num, {
                        ...existing,
                        current: e.payload.current,
                        total: e.payload.total,
                    });
                }
            })
        );
        unlisteners.push(
            await events.volumeCompleteEvent.listen((e) => {
                const existing = volumes.get(e.payload.volume_num);
                if (existing) {
                    volumes = new Map(volumes).set(e.payload.volume_num, {
                        ...existing,
                        done: true,
                        success: e.payload.success,
                    });
                }
            })
        );
        unlisteners.push(
            await events.conversionCompleteEvent.listen((e) => {
                duration = e.payload.duration_secs;
                status = 'done';
            })
        );

        // Start converting immediately
        status = 'converting';
        try {
            await commands.convert(
                {
                    output_dir: wizard.outputDir,
                    output_name: wizard.outputName,
                    create_directory: wizard.createDirectory,
                    image_format: wizard.imageFormat as 'Avif' | 'Webp' | 'Original',
                    max_width: wizard.maxWidth ?? undefined,
                    output_format: wizard.container as 'Cbz' | 'Epub' | 'Raw',
                    direction: wizard.direction as 'Ltr' | 'Rtl',
                    bundle: wizard.bundle === 'auto' ? 'auto' : 'flatten',
                    volume_separator: wizard.volumeSeparator,
                    hide_single_volume: wizard.hideSingleVolume,
                },
                wizard.pageEdits.map((vol) => ({
                    volume_num: vol.volumeNum,
                    pages: vol.pages.map((p) => ({
                        original_page_index: p.originalPageIndex ?? undefined,
                        custom_path: p.customPath ?? undefined,
                        excluded: p.excluded,
                    })),
                }))
            );
        } catch (e) {
            status = 'error';
            errorMessage = String(e);
        }
    });

    onDestroy(() => {
        unlisteners.forEach((u) => u());
    });
</script>

<h2>Converting…</h2>

{#if status === 'converting' || status === 'done'}
    {#each [...volumes.entries()] as [num, vol]}
        <div style="margin-bottom:12px;">
            <div style="display:flex;justify-content:space-between;margin-bottom:4px;">
                <span>{vol.name}</span>
                <span
                    style="color:{vol.done && vol.success
                        ? '#10b981'
                        : vol.done
                          ? '#ef4444'
                          : '#6b7280'};"
                >
                    {vol.done
                        ? vol.success
                            ? '✓ Done'
                            : '✗ Failed'
                        : `${vol.current}/${vol.total}`}
                </span>
            </div>
            <div style="background:#374151;border-radius:4px;height:8px;">
                <div
                    style="background:#6366f1;border-radius:4px;height:8px;width:{vol.total
                        ? (vol.current / vol.total) * 100
                        : 0}%;transition:width .3s;"
                ></div>
            </div>
        </div>
    {/each}
{/if}

{#if status === 'done'}
    <p style="color:#10b981;">All done in {duration.toFixed(1)}s!</p>
    <button
        onclick={() => {
            wizard.reset();
            goto('/');
        }}>Start over</button
    >
{/if}

{#if status === 'error'}
    <p style="color:#ef4444;">Error: {errorMessage}</p>
    <button onclick={onBack}>← Back</button>
{/if}

<script lang="ts">
    import { wizard } from '$lib/wizard/state.svelte';
    import { onMount } from 'svelte';

    const KEY = 'thasia:settings';

    type Defaults = {
        imageFormat: 'avif' | 'webp' | 'original';
        container: 'cbz' | 'epub' | 'raw';
        direction: 'ltr' | 'rtl';
        bundle: 'auto' | 'flatten';
        volumeSeparator: string;
        hideSingleVolume: boolean;
        createDirectory: boolean;
        maxWidth: number | null;
    };

    let defaults = $state<Defaults>({
        imageFormat: 'avif',
        container: 'cbz',
        direction: 'ltr',
        bundle: 'auto',
        volumeSeparator: ' - ',
        hideSingleVolume: false,
        createDirectory: false,
        maxWidth: null,
    });

    let saved = $state(false);

    onMount(() => {
        const raw = localStorage.getItem(KEY);
        if (raw) {
            try {
                Object.assign(defaults, JSON.parse(raw));
            } catch {}
        }
    });

    function save() {
        localStorage.setItem(KEY, JSON.stringify(defaults));
        // Apply to current wizard state too
        wizard.imageFormat = defaults.imageFormat;
        wizard.container = defaults.container;
        wizard.direction = defaults.direction;
        wizard.bundle = defaults.bundle;
        wizard.volumeSeparator = defaults.volumeSeparator;
        wizard.hideSingleVolume = defaults.hideSingleVolume;
        wizard.createDirectory = defaults.createDirectory;
        wizard.maxWidth = defaults.maxWidth;
        saved = true;
        setTimeout(() => (saved = false), 2000);
    }
</script>

This may need adjustment depending on how `start_pipeline` resolves image data — it calls `source.fetch(&parsed.source)` which reads from `parsed.source.absolute_path`, so a minimal `LocalSource` pointing at any directory will work since fetch reads the absolute path directly
